# QML implementation of [Eduport](https://eduport.webestica.com)

# Installation

## Getting

```sh
$ git clone git://darapsa.org/qeduport.git
$ cd qeduport
$ mkdir build
$ cd build
```

## Installation

```sh
$ /Applications/Qt/QtDesignStudio/qt6_design_studio_reduced_version/bin/qmake ../Eduport.pro
$ make install
```
